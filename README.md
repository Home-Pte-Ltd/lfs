# lfs


## add files to LFS
1. go to repo and do git lfs install
2. add files to lfs
- git lfs track "*.png" for example to track all files with extension of .png
3. add lfs files
- normal git add process. 
4. view lfs files staged
- git lfs status
5. view lfs files
- git lfs ls-files
